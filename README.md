# aischedule

小爱课程表适配代码收集仓库

本项目为 [复活吧，小爱课程表](http://xiaoai.06xy.cn) 的适配代码集中与维护仓库，欢迎各位开发者提交适配方案，共同完善生态。

注意：项目是赶工出来的，暂时没有写网站自动拉取github项目的代码，目前还是作者手动上传，会有一些时间差，轻谅解。

（作者也是一个小白，不太会很多操作，如有问题，敬请指出，不胜感激。）

---

## 📦 提交内容要求

请确保提交以下四个文件：

* `parser.js`
* `timer.js`
* `provider.js`
* `info.txt`

---

## 📝 info.txt 格式规范

`info.txt` 请严格按照以下格式填写：

```
适配学校：xxx
项目名称：xxx
开发者昵称：xxx
开发者QQ：xxxx
项目简介：xxx
个人留言（可选）：xxxx
```

### 示例

```
适配学校：河北地质大学
项目名称：URP接口版
开发者昵称：颜致恒
开发者QQ：2488971290
项目简介：登录后直接点击一键导入，适配新版 URP 系统
个人留言（可选）：希望可以帮到更多同学
```

---

## 📂 提交目录规范

请将文件提交至以下目录结构：

```
code/学校名称/项目名称
```

### 示例

```
code/河北地质大学/URP接口版
```

请确保目录名称与 `info.txt` 中填写的“适配学校”和“项目名称”保持一致。

---

## ⚠ 注意事项

* 文件命名必须严格按照规范。
* 请确保代码可正常运行并已测试。
* 项目简介应简洁明了，清晰说明适配内容
